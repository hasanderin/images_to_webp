Is the Bexio Odoo connector compatible with Odoo Enterprise and as a Community?

Yes, the Bexio Odoo connector is built to work perfectly with Odoo Enterprise (On-premise and Odoo.SH) as well as the Community. The app, however, cannot be installed on Odoo Cloud, because the Cloud doesn’t support any third-party applications.

I need an extra feature for the Odoo Bexio connector. Can I request it?

Yes, you can request a feature to be added to the application. Contact us to request customization.

Will I get support and free upgrades if I purchase the app?

In the case of a purchase, you will get free updates and support for the first month. After it, you’ll need to subscribe to our app in the Bexio marketplace to continue getting updates and support.

How many Bexio instances can be connected with a single Odoo instance?

You can connect as many Bexio instances as you may want or need. There is no limit.

Is only a one-way synchronization possible?

Yes. The application allows synchronizing the data one way. You can set up your synchronization preferences in the settings and choose the option that fits you best.
